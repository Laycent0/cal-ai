# Cal AI — Telegram Nutrition Bot 🤖🔥

Бесплатный Telegram-бот, который:
- Подбирает персональные КБЖУ под твою цель (похудение / набор мышц / набор веса / поддержание)
- Анализирует фото еды через Gemini Vision AI
- Ведёт дневник питания и счётчик воды

## 🧮 Как считается КБЖУ

**Формула Mifflin-St Jeor:**
- Мужчины: `BMR = 10×вес + 6.25×рост - 5×возраст + 5`
- Женщины: `BMR = 10×вес + 6.25×рост - 5×возраст - 161`

**TDEE = BMR × коэффициент активности**

**Корректировка на цель:**
| Цель | Калории |
|------|---------|
| Похудеть (медленно) | TDEE - 10% |
| Похудеть (умеренно) | TDEE - 20% |
| Похудеть (быстро)   | TDEE - 25% |
| Набор мышц (новичок)| TDEE + 12% |
| Набор мышц (средний)| TDEE + 8%  |
| Набор мышц (проф)   | TDEE + 5%  |
| Набор веса (умеренно)| TDEE + 15% |
